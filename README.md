# MamaCare
It is a pregnancy care care app that tracts nutrition, suggest food there is also the feature to add partner.
## Features
1. Nutrition calculator
2. Food suggestion
3. Add your parner
4. Event tracker

```bash
git clone https://github.com/RUBY-DANGOL/MamaCare.git

npm install
```
